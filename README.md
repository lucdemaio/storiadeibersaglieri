# 🎖️ Storia dei Bersaglieri - Sito Web Moderno 2026

Un sito internet spettacolare dedicato alla storia, tradizione e gloria dei Bersaglieri italiani. Progettato con uno stile moderno 2026, ricco di dettagli storici approfonditi e animazioni affascinanti.

## 📋 Contenuti Principali

### 🏛️ La Storia Leggendaria
- **1836**: La fondazione del corpo da parte di Carlo Alberto di Savoia
- **1848-1870**: L'Indipendenza Italiana e le campagne risorgimentali
- **1866**: La Battaglia di Custoza durante la Terza Guerra d'Indipendenza
- **1896**: La Campagna d'Africa - Battaglia di Adwa
- **1915-1918**: La Grande Guerra - Oltre 180.000 Bersaglieri coinvolti
- **1940-1945**: La Seconda Guerra Mondiale
- **1946-2026**: La Repubblica Italiana e l'eredità moderna

### ⚔️ Le Battaglie Memorabili
- **La Presa di Roma** (20 Settembre 1870) - L'evento definitivo dell'unificazione italiana
- **Castelfidardo** (18 Settembre 1860) - Sconfitta definitiva dell'esercito papale
- **La Battaglia dell'Isonzo** (1917) - Una delle campagne più dure della Grande Guerra
- **Adwa** (1896) - La campagna coloniale africana
- **Custoza** (1866) - Battaglia onorevole durante la Terza Guerra d'Indipendenza

### 🎺 Le Fanfare Leggendarie
- Marcia dei Bersaglieri - La marcia più famosa della storia militare italiana
- I Tamburi di Guerra - Coordinamento e unità nei momenti cruciali
- Le Trombe della Carica - Il suono che ha preceduto gli attacchi gloriosi
- Inni e Marce - Celebrazioni solenni della tradizione

### 👥 I Raduni dei Bersaglieri
- Il Raduno Nazionale - Migliaia di Bersaglieri si riuniscono ogni anno
- Raduni Regionali - Celebrazioni territoriali in ogni regione italiana
- Raduni Generazionali - Incontri tra leve di soldati
- Partecipazioni Internazionali - Bersaglieri nel mondo
- Raduni Tematici - Celebrazioni speciali di anniversari importanti
- Raduni Familiari - L'eredità tramandata di generazione in generazione

### 🌟 L'Eredità Permanente
- Il Simbolismo Nazionale
- L'Innovazione Militare
- L'Onore e la Disciplina
- La Cultura Popolare
- Statistiche impressionanti: 190+ anni di tradizione, 1000+ Medaglie d'Oro, 500.000+ arruolati nella storia

## 🚀 Come Usare

1. Scarica tutti i file nella stessa cartella
2. Apri `index.html` nel tuo browser web preferito
3. Naviga attraverso le sezioni usando i link nella navbar
4. Goditi le animazioni e gli effetti interattivi

### Deployment Online
Per deployment online:
1. Carica i file su un web server (es. Netlify, Vercel, GitHub Pages)

## 🔧 Personalizzazione

### Cambiare i Colori
Modifica le variabili CSS in `styles.css`:
```css
:root {
    --primary-color: #DC143C;  /* Colore primario */
    --secondary-color: #001A80;  /* Colore secondario */
    --accent-color: #FFD700;  /* Colore accento */
    /* ... altri colori */
}
```

### Aggiungere Nuove Sezioni
1. Aggiungi il nuovo `<section>` in `index.html`
2. Crea lo stile CSS corrispondente
3. Aggiungi il link nella navbar
4. Aggiungi scroll trigger nel `script.js`

### Modificare Contenuto
Tutti i testi storici sono direttamente nell'HTML e possono essere facilmente modificati o espansi.
